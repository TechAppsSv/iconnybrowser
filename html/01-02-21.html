MonocromatixV4 Conoce Sus Novedades

Escrito por Admin 01/02/2021



Recientemente hemos liberado la interfaz escrita css para navegadores que usen electron-navigation/WDK que ha sido MonocromatixV4 (Version 4) y
Iconny utiliza esta interfaz que es muy sencilla por el momento Iconny utiliza MonocromatixV2 en CupCake 0.09 entonces ahora usaremos MonocromatixV4 en nuestro proximos lanzamiento
este ya ha sido implementado en Gelato 1.00 esta interfaz lleva 3 meses de desarrollo

Novedades

  • Como primera novedad es que esta vez MonocromatixV4 depende menos del uso de Boostrap ahora monocromatix usa su propios estilos para div, input, ahref, label y button

  • Como segunda novedad es que los colores han cambiado a negro como fondo y un gris bajo como segundo color y como tipografia el color es blanco y sigue usando las fuentes de google en este caso es Poppins
  • Tercera novedad es que su estilo de botones posee efectos de varios colores
  • Cuarta Novedad los inputs ya no usan bootstrap y ahora posee un estilo propio y elegante
  • la interfaz solo posee tres colores blanco, gris y negro
  • Se ha rendondeado la barra de links
  • La fecha y hora se encuentra abajo
  • Las pestañas han sido cambiadas de posicion y ahora va abajo de la barra de links
  • Las pestañas ahora son mas largas y mas delgadas
  • El dropdown va arriba de la barra de links
  • Integracion con web panels para WDK
  • Mejor Optimizado
  • Mejor consistencia del diseño
  • Se hace mas el uso de iconos de Font font-awesome
  • Ahora se usa los iconos de Material Icons

    • Esta version tendra soporte de actualizaciones hasta el 1 de abril y soporte extendido de bugs hasta el 1 de septiembre (esto en caso que no haya salido una nueva version para el momento)
    Ademas la integracion con Iconny sera introducido con Gelato usando MonocromatixV4 por el momento no hemos integrado esto en la version light y solo funciona bien en la version dark
    para el uso de MonocromatixV4 es necesario el uso de google-fonts, material-icons, bootstrap, electron-navigation y WDK 1.01 LTS o WDK 1.03 Stable
    MonocromatixV4 se puede descargar desde el codigo fuente de Iconny
    Los cambios de Iconny solo son que se ha actualizado la version de ElectronJS 11.2.1 ademas que se desahabilito darkmode-js y el modo oculto dark
    y se ha aplicado la interfaz de MonocromatixV4 ha todas las opciones menos WDK
    Por ultimo se han corregido bugs del webview principal y se confirma el buen funcionamiento el deepin 20.1, pop os 20.10 , ubuntu 20.10 y ubuntu 20.04.1 LTS


    Esperamos que les guste esta nueva interfaz que es el objetivo de Gelato Introducir una mejor interfaz
    Se ha dedicado mucho esfuerzo en esta interfaz que es la primera novedad de Gelato
    En las proximas horas se actualizara todos los repositorios de descargas
