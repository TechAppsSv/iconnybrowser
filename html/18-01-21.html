Iconny Browser Linux "CupCake" 0.09 Stable

Escrito por Admin 18/01/2021



Despues de 7 dias de desarrollo y pruebas ademas dos Betas nos alegra anunciar "CupCake" Stable 0.09 como se dijo en anteriores notas CupCake quiere complementar
a Coffe Cake haciendolo mas completo en funciones basicas y necesarias y no tanto funciones nuevas ademas otros de los objetivo de CupCake es unificar y mejorar la interfaz ademas de mejorar la paleta de colores
y se trato de cumplir lo mejor posible el calendario para que no llegara atrasado este lanzamiento

Novedades

Como primera novedad es que se ha cambiado ligeramente la paleta de colores en el darkmode siendo mas un poco de tonalidad entre negro y azul


Rediseño


Como segunda es que el boton para mas opciones desaparecio siendo reemplazado por un dropdown para abrir las demas opciones


Dropdown


Tercera novedad es que se incluye una hora y una fecha fija en el navegador

Fecha Y Hora Fija





Cuarta novedad es que se añadio el Inspector que sirve para inspeccionar paginas web con la DevTools de Chromium

Inspector







Quinta novedad ahora cuando se elige el modo claro ya Notely cuenta con una version sin interfaz oscura

Notely interfaz clara

Como sexta es que se ha incorporado un Menu Contextual que tendra el diseño del tema GTK o dependiendo del diseño nativo del sistema
en el que se use este tiene las funciones basicas esenciales

Menu Contextual Nativo


Como septima novedad es que ahora se incluye una version especialmente hecha para los web panels de Notely


Notely En Web Panels


Octava novedad es que desde Notely para web panels se abre en una ventana los links esto es asi para hacer webapps

WebApps Con Notely


Los repositorios de Iconny Updates y el repositorio principal seran actualizados en las proximas horas para descargar "CupCake"

Lastimosamente por el momento todavia no se tiene lista la version snap

Ademas un cambio que no puede estar en las seccion de cambios menores es que el buscador de URL utiliza DuckDuckgo esto hemos hecho por los bangs
Consideramos que esta es una funcion muy interesante y por cual motivo gracia estos se puede buscar en una variedad de buscadores
por ejemplo para buscar en google !g (inserte busqueda) o !gi para buscar en GitHub

Otros cambios menores :
  • Correcion de errores los botones de modo divido para cerrar no se miraban bien
  • Se actualizo el tamaño de los webview con el controlador escrito en CSS
  • Correcion el bloqueador de anuncios no funcionaba
  • Se actualizo ElectronJS de la version 11.1.1 a la version 11.2.1
  • Se actualizo el bloqueador de anuncios @cliqz/adblocker-electron de la version 1.18 a la version 1.19
  • Correcion de errores en WDK 1.01 LTS en tipografias
  • Ahora se usa el icono predeteminado del electron-navigation
  • Notely For WebPanels ahora cuenta con el modo claro
  • Icono de mas opciones en la barra se miraba a veces
  • Ahora WDK DevTools cuenta con un modo claro (esta es la primera modificacion no oficial al proyecto de WDK esta solo esta disponible en Iconny porque el equipo oficial de WDK que
    es Codeny Dev no ha hecho el commit de esta modificacion al proyecto WDK )
  • Correcion de tipografias de WDK en DevTools
  • Se cambio el color gradiante de la pagina de inicio
  • Ahora cuando se abre la pagina de nueva pestaña se puede acceder al blog
  • Se mejoro la paleta de colores en el modo oscuro haciendolo mas unificada
  • Se actulizo los links del centro de descargas
  • Se esta preparando la version ya compilada de Iconny para Windows y Mac y Snap



    • Esperamos que les guste estos nuevas funciones y cambios ya que hasta el momento esta ha sido la actualizacion mas grande del navegador
    Se ha dedicado mucho esfuerzo en esta version
